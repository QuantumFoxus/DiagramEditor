# Demo Diagram Editor

## Описание

Demo Diagram Editor — это приложение на JavaFX для создания, редактирования и сохранения диаграмм. Программа поддерживает добавление различных блоков, линий и других элементов, а также экспорт диаграмм в PNG и сохранение/загрузку в собственном формате.

Проект задуман как учебный, в рамках сдачи курсовой работы, но будет улучшаться для удобства работы с диаграммами.

## Основные возможности

- Добавление и редактирование элементов диаграммы (блоки, линии и др.)
- Перемещение и изменение размера элементов
- Отмена и повтор действий (Undo/Redo)
- Сохранение и загрузка диаграмм в формате `.dia`
- Экспорт диаграммы в PNG
- Масштабирование и настройка сетки

## Требования

- Java 17+
- Maven 3.6+
- JavaFX 21.0.2

## Сборка и запуск

1. Склонируйте репозиторий:
   ```sh
   git clone <repo-url>
   cd demo
   ```

2. Соберите проект с помощью Maven:
   ```sh
   mvn clean package
   ```

3. Запустите приложение:
   ```sh
   mvn javafx:run
   ```

## Структура проекта

- `src/main/java` — исходный код приложения
- `src/main/resources` — ресурсы (FXML, иконки и др.)
- `runtime/` — настройки среды выполнения и лицензии
- `pom.xml` — конфигурация Maven
